Data Structure & Algorithms

Project Overview

Through my Data Structures and Algorithms project, I have explored a wide range of fundamental and advanced algorithms. This journey has taken me from the basics—such as arrays, linked lists, stacks, and queues—to more complex topics like trees, graphs, sorting, searching, and dynamic programming. By implementing and analyzing these algorithms, I have gained a deeper understanding of how data is organized, manipulated, and optimized in computer science. This project has strengthened my problem-solving skills and provided a solid foundation for tackling real-world programming challenges.

Topics Covered

  • Arrays, Linked Lists, Stacks, Queues
  • Trees (Binary Trees, BST, AVL, etc.)
  • Graphs (BFS, DFS, Shortest Path)
  • Sorting & Searching Algorithms
  • Recursion & Dynamic Programming
  • Hash Tables & Heaps
  • Algorithm Analysis & Optimization

Sample Implementations

  • Binary Search Tree Visualizer – Interactive tool to insert, delete, and search nodes.
  • Sorting Algorithm Demo – Visual comparison of bubble, merge, and quick sort.
  • Graph Traversal Simulator – BFS and DFS step-by-step animations.
